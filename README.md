# matcha

This project is a website where I get to express my love for matcha, other fun drinks and food. My original goal was to create an aesthetically pleasing website that reflects what I like. I wanted it to feel fun, aesthetic and similar to real review vlogs. My goal was to build a fully functional website from scratch using HTML and CSS that includes mutiple pages, styled layouts, and organized content sections. I also wanted to learn GitHub.

What my project is/does:
My project is a website about reviewing food and matcha. It's a site where I post reviews of matcha drinks and food, with different pages users can click through.

Why I made it:
I made this because I like matcha and food and drinks. I wanted to create something fun instead of a basic boring project. I also wanted to learn how websites are actually made.

How I made my project:
I used HTML to build the structure and css to design it. I worked in vs code and connected to github. I created multiple pages and styled things like images,borders and layouts. 

WHat I struggled with and what I learned:
I struggled with getting files uploaded correctly and fixing layouts in css. It was frustrating at times, but I learned how to fix it step by step. I also learned important small design details are.

Reflection:
This project helped me understand how websites work and made me better at coding and problem solving.
